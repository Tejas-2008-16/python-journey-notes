## 🛠️ What's inside each Day folder?
 
Every day has **2 files:**

---
 
## 🚀 How to run the practice code
 
1. Make sure Python is installed → [python.org](https://python.org)
2. Open your terminal
3. Go to the day's folder and run:
```bash
python practice.py        # Windows
python3 practice.py       # Mac / Linux
```
